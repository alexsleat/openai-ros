# openai-ros
ChatGPT + ROS wrapper

## Configure:
Create a new file called config.sh with your, eg:

    export OPENAI_API_KEY="xx-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX"

## Building the docker:
    . build_docker.sh

## Run the docker:
    . run_docker.sh
    
## Attach a new terminal to the running docker:
    . run_docker.sh
